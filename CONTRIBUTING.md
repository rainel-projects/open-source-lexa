# Contributing to LEXA

We welcome contributions to Lexa! Whether you're adding a new refactoring rule, a language parser, or improving the UI, your help is appreciated.

## How to Contribute

1. **Fork the repo** and create your branch from `main`.
2. **Setup your environment** (see README.md).
3. **Add tests** for your changes.
4. **Pass all checks**: `cargo test`, `npm test`.
5. **Submit a Pull Request**.

## Rule Definitions
Rules are defined in JSON in `core/rules`. See `docs/rules-spec.md` for the schema.

## Code Style
- **Rust**: `cargo fmt`
- **TypeScript**: Prettier & ESLint

Thank you for building with us!
